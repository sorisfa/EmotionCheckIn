# EmotionCheckIn\n\nThis React TypeScript project is designed to facilitate emotional check-ins for students. Users can select their mood and check-in using various methods, as well as upload data and view a dashboard.\n\n## Installation\n\nTo run the project, use the following commands:\n\n```bash\n# Clone the repository\ngit clone https://github.com/sorisfa/EmotionCheckIn.git\ncd EmotionCheckIn\n\n# Install dependencies\nnpm install\n\n# Start the project\nnpm start\n```\n\n## License\n\nThis project is licensed under the MIT License.
